# Collection Exercises

In this exercise, you will learn how to work with Rust standard collections.

## After completing this exercise you are able to

- Use the Rust `std::vec::Vec` dynamic array container.
- Sort sequences.
- Use the Rust `std::collection::BTreeSet` sorted set container.
- Use the `std::collections::HashMap` map container.
- Implement a stack and a queue using the `std::vec::VecDeque` container.

## Task 1

Read in a string of space separated words into a vector. Then sort the vector and print the
sorted list.

## Task 2

Perform the same task as in task 1, but use the `std::collections::BTreeSet` container instead
which keeps the elements in sorted order.

## Task 3

Read in a string with the syntax "key1: value1, key2: value2" and store the key-value pairs in a
hash map.
